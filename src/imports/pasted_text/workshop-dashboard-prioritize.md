Create a professional workshop dashboard interface displayed on a large TV screen in a meeting room.

The interface represents a live UX workshop system that collects real-time feedback using physical interactive tools.

Style:
Clean, structured, enterprise-level design.
Vector-based layout.
Minimal color usage.
Professional research-lab aesthetic.
Not playful, not startup flashy.
Balanced whitespace.
Grid-based layout.

Left:
Workshop Name:

Below that, a small agenda card:

Title:

Live Group State
